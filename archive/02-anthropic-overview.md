# The Anthropic Tools

Anthropic has a mix of tools. Each has a center, but the borders are fuzzy.

**Claude Chat** -> **Claude Cowork** -> **Claude Code**

Looking deeper.

**Claude Chat (web)** is most similar to ChatGPT Web. Most people interact with it by pasting context into the window, and it doesn't have access to your files. (Can connect to Web Tools)

**Claude Browser Plugin** Allows Claude to (very slowly aand awkwardly) explore and manipulate websites.

**Claude Chat (Desktop)** has the same interface (but can connect to some local tools)

**Claude Cowork** is billed as the no-code workflow tool. It also connects to web and local tools, but expands from chats to "projects" and adds triggering and managing workflows.

**Claude Code (Web)** is a very powerful sandboxed coding tool, that usually works on a Github Repository. You get few security prompts because it's not managing your files, and you can always roll back.

**Claude Code (Desktop or CLI)** is the start of the most powerful tool, adding the ability to work inside a folder and/or a Github Repo.

**Claude Code (IDE)** is the most powerful tool. You can manage your files in one area and manage Claude in other areas. The sky is the limit. You can build a basic tool or full software.

## Permissions

Every level has its own level of permissions. You can decide when Claude will ask permission. Maybe Read-only opeations are permitted, but writes require permission.
