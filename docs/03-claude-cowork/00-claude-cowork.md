# Claude Cowork

## What's Different from Claude Chat

If you're used to Claude Chat, Cowork feels similar at first — but the mental model is different.

| | Claude Chat | Claude Cowork |
|---|---|---|
| **Mode** | Conversation | Working session |
| **File access** | Upload only (20 files, 30MB each) | Direct access to your local folders |
| **Output** | Text you copy-paste | Actual files created/edited on your machine |
| **Tasks** | Single-turn or short exchanges | Multi-step plans that run to completion |
| **Memory** | Cloud-stored, limited | Saved to files on your computer |

The key shift: **Chat is for thinking. Cowork is for doing.**

Give Cowork a goal and a folder, and it reads, creates, and edits files until the job is done. You steer along the way rather than managing every step.

## Getting Started

1. Open Claude Desktop and click the **Cowork** tab in the mode selector
2. Grant Cowork access to a specific folder on your computer
3. Describe a task — Cowork will plan steps and ask before taking significant actions

## Key Concepts

### Projects

Group related tasks into separate workspaces, each with their own files, context, instructions, and memory. Think of a project as a persistent working environment for a specific topic or goal.

### Plugins

Plugins bundle skills, connectors, and sub-agents into a single package customized for a role or workflow. Browse available plugins at [claude.com/plugins](https://claude.com/plugins).

### Connectors

Link Cowork to external tools:

- **n8n** — workflow automation
- **AWS Marketplace** — cloud resources
- **Honeycomb** — observability data
- **Fellow.ai** — meeting insights

You can also add custom connectors for tools not on the list.

### Scheduled Tasks

Type `/schedule` in any Cowork task to set it up to run automatically on a schedule or on demand.

## Typical Use Cases

- **File organization** — point at a folder of drafts or downloads; ask Cowork to rename, sort, or deduplicate
- **Document drafting** — hand off source files and get a structured draft back, ready to refine
- **Research workflows** — Cowork reads across many files and surfaces what's relevant
